                     
                     
                     # 🌿 Hoomie's – Skincare E-Commerce Website

  A modern and responsive skincare e-commerce website built using **HTML, CSS, and JavaScript**. The project focuses on providing a clean shopping experience with an intuitive user interface, dynamic product pages, category browsing, and a fully functional shopping cart.



✨ Features

* 🏠 Modern and responsive homepage
* 🛍️ Product catalog with multiple skincare products
* 🔍 Product search functionality
* 📂 Category-wise product browsing
* 📄 Individual product details page
* ⭐ Product ratings and descriptions
* 🛒 Shopping cart with quantity management
* 💰 Automatic price calculation
* 📱 Responsive design for desktop and mobile
* 🎨 Minimal green & white theme inspired by premium skincare brands



🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)

🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/shravankumawat17/Ecommerce-website.git
```

2. Open the project folder.

3. Launch `index.html` in your browser.

No additional installation or dependencies are required.
